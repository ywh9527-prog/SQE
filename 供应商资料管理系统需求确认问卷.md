## 📁 二、资料分类管理

### 2.1 资料类型定义
**请确认SQE管理的质量资料类型：**

- [30天预警] **质保协议** (Quality Agreement)
  - 必填字段：协议编号、签订日期、到期日期、责任人
  - 文件格式：PDF / Word / Excel
  - 预警提前期：30天 / 60天 / 90天

- [ROHS和REACH,30天预警， MSDS作为单独一项] **环保资料** (Environmental Documents)
  - 包含：ROHS、REACH、MSDS等
  - 必填字段：证书类型、证书编号、有效期、发证机构
  - 预警提前期：30天 / 60天

-  []**CSR资料** (CSR Documents)
  - 包含：社会责任报告、合规证书等
  - 必填字段：报告年度、认证范围、有效期
  - 预警提前期：30天 / 45天

**说明：采购相关资料（购销合同、商务协议）不在本系统管理范围内**

### 2.2 文件管理策略
**请确认文件存储和管理方式：**

- [最好是可以支持预览] 文件预览：支持PDF预览 / 图片预览 / 不支持预览
